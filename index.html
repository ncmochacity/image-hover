Filter VS. Find

What's the difference between the filter and find methods? Filter() works on the currently matched elements by reducing the set of matched elements to those that pass the function's test, by constructing new jQuery object with the subset of matching items. Find() will get the descendants of each element in the current set of matched elements, filtered by a selector, jQuery object, or element.
